Netflix Jikkyo is a Chrome extension that overlays Niconico-style live comments (danmaku) on Netflix in real time. Share the moment with everyone watching the same title.

- Peer-to-peer — no central server, your comment data stays in your browser
- Zero configuration — just install and start watching

## Features

- **Live comments** — Scrolling danmaku comments shared in real time with other viewers
- **Side panel** — View comment list and title info in Chrome's sidebar
- **NG filters** — Block specific comments or users
- **Multilingual** — Japanese / English / Korean / Czech / Traditional Chinese
- **Customization** — Adjust speed, opacity, font, and size
- **Pop-out** — Detach the side panel into a standalone window

## How to Use

1. **Play a title on Netflix** — The extension automatically connects you with other viewers
2. **Type a comment** — Use the input bar at the bottom of the player
3. **Manage in the side panel** — Open Chrome's sidebar to see the comment list and settings

## Privacy

Comment data is stored only in your browser. Nothing is sent to external servers and no browsing data is collected.
